## Description
This project is an interactive web application designed to help individuals generate personalized career roadmaps within the cloud engineering and tech industry. Leveraging AI, it analyzes user input to suggest skills, projects, and learning paths.

---

## Features

*   **Personalized Roadmaps:** Generates tailored career paths based on user aspirations and current skill level.
*   **AI-Powered Skill Suggestions:** Recommends relevant skills and technologies for specific career goals.
*   **Project Ideas:** Provides actionable project suggestions to build a strong portfolio.
*   **Interactive Interface:** User-friendly interface for easy navigation and input.
*   **Tech Industry Focus:** Primarily targets careers in cloud engineering and related tech fields.

---

## Tech Stack

*   **Languages:** TypeScript
*   **Frontend:** React, Vite
*   **AI Integration:** Google Gemini API

## Installation

**Prerequisites:** Node.js, npm/yarn

1.  **Clone the repository:**
    ```bash
    git clone [repository-url]
    cd cloud-engineering-career-roadmap-3
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```
3.  **Set up environment variables:**
    Create a `.env` file in the root directory and add your Google Gemini API key:
    ```
    GEMINI_API_KEY=your_gemini_api_key_here
    ```
    *Note: A `.env.local` file might be used for local development, ensure it's added to `.gitignore`.*

4.  **Run the development server:**
    ```bash
    npm run dev
    # or
    yarn dev
    ```

---

## Usage

Once the development server is running, open your browser to `http://localhost:5173` (or the port specified by Vite). Follow the on-screen prompts to input your career goals and generate your personalized roadmap.
