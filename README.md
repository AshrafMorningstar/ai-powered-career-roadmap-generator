## Description

This project is an AI-powered web application designed to help individuals navigate their technology career paths. By leveraging AI, it generates personalized roadmaps, suggests relevant skills, and provides insights into career development.

---

## Features

- **Personalized Roadmap Generation:** Get a tailored career path based on your goals.
- **Skill Identification:** Discover essential skills for your desired roles.
- **AI-Driven Insights:** Utilize AI to understand career trends and development.
- **Interactive User Interface:** An intuitive and responsive interface for easy navigation.

---

## Tech Stack

- **Languages:** TypeScript
- **Frontend:** React, Vite
- **AI Integration:** Google Gemini API

## Installation

**Prerequisites:** Node.js and npm/yarn installed.

1.  Clone the repository:
    ```bash
    git clone [repository-url]
    cd ai-powered-career-roadmap-generator
    ```
2.  Install dependencies:
    ```bash
    npm install
    # or
    yarn install
    ```
3.  Create a `.env.local` file in the root directory and add your Gemini API key:
    ```
    GEMINI_API_KEY=your_gemini_api_key_here
    ```

---

## Usage

To run the application locally:

```bash
npm run dev

yarn dev
```
